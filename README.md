# Simple Form App

Одностраничное приложение на React с тремя последовательными формами для сбора данных пользователя и финальным модальным окном подтверждения.

## Функциональность

1. **Форма 1 – Личные данные**
   - Телефон (маска 0XXX XXX XXX)
   - Имя, Фамилия
   - Пол (Мужской / Женский)
   - Валидация всех полей

2. **Форма 2 – Адрес и место работы**
   - Выбор места работы из списка, полученного по API (dummyjson)
   - Адрес проживания
   - Кнопка «Назад» сохраняет данные

3. **Форма 3 – Параметры займа**
   - Ползунок суммы ($200–$1000, шаг 100)
   - Ползунок срока (10–30 дней, шаг 1)
   - Ручной ввод через стрелки
   - Склонение слова «день»
   - Отправка тестового POST-запроса и модальное окно с результатом

4. **Защита маршрутов** – нельзя попасть на форму 2 или 3, не заполнив предыдущие.

## Установка и запуск

# Клонировать репозиторий

git clone https://github.com/AlekseyPerf/simple-form-app.git

# Перейти в папку проекта

cd simple-form-app

# Установить зависимости

npm install

# Запустить в режиме разработки

npm start
